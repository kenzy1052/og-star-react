## Installation & Running

1. Navigate to the project directory:
   ```bash
   cd ogstar-updated
   ```

2. Install dependencies (if not already installed):
   ```bash
   npm install
   # or
   bun install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   bun dev
   ```

4. Build for production:
   ```bash
   npm run build
   # or
   bun run build
   ```
